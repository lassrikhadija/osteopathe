# Lumen Ostéopathie — site démo (modèle pour ostéopathes)

Site vitrine **de démonstration** pour une clinique d'ostéopathie, conçu par **[NEXTIWEB Studio](https://nextiweb.ca)**.

> ⚠️ **Clinique fictive.** « Lumen Ostéopathie » n'existe pas : coordonnées, praticienne, statistiques, avis et tarifs sont factices. Ce dépôt sert d'échantillon de portfolio et de modèle réutilisable pour les ostéopathes.

## Contenu

- `index.html` — version française (FR-CA)
- `index-en.html` — version anglaise (EN-CA)
- `sitemap.xml`, `robots.txt`, `llms.txt` — fichiers SEO / IA (standard 2026)

## Caractéristiques

- **Design hi-tech 2026** — clair & lumineux · crème · anthracite · gris-rose · touche bordeaux · cuivre rosé
- **Bilingue** FR/EN avec `hreflang` réciproques et `canonical`
- **UX calibrée sur les douleurs du persona** (mal de dos / télétravail, migraines & stress, grossesse, sportif, parent de nourrisson)
- **Lumière qui suit le curseur** (désactivée sur mobile et si `prefers-reduced-motion`)
- **Conforme Loi 25** — bandeau de consentement aux témoins, témoins essentiels uniquement, aucun suivi publicitaire tiers
- **Accessibilité** — lien d'évitement, navigation clavier, contrastes conformes, reveals « fail-open »
- **Page autonome** — un seul fichier HTML par langue (CSS + JS inline), aucune dépendance

## Domaine cible

`https://osteopathe.nextiwebstudio.ca/`

## Objectif (au service de NEXTIWEB, sans concurrence)

Cette démo **dessert** la page commerciale **[https://nextiweb.ca/osteopathes.html](https://nextiweb.ca/osteopathes.html)**. Elle est indexable (`index,follow`) et **tous les liens sortants vers NEXTIWEB sont en dofollow** afin de transmettre l'autorité SEO à cette page. Le but est de **servir NEXTIWEB, pas de la concurrencer** : la démo se présente partout (HTML, JSON-LD, llms.txt) comme une vitrine NEXTIWEB et renvoie humains et IA vers `nextiweb.ca/osteopathes.html`.
